# 更新日志

本项目的主要变更记录在此文件中。版本号遵循[语义化版本](https://semver.org/lang/zh-CN/)。

## [1.1.0] - 2026-07-13

### 新增

- 新增绿色建筑评价、绿色建筑验收、绿建评价、绿建验收、绿色建筑等保留规则。
- 新增结构监测、运营期结构监测、监测服务等监测类标题保留规则。
- 筛选后 Excel 新增“筛选删除记录”工作表，保留被终筛删除的项目和筛出原因。
- 桌面界面新增运行进度反馈：滚动进度条、状态栏摘要和列表/详情/写入阶段日志。
- 新增浏览器自动检测与专用登录环境，支持 Chrome、Brave、Edge 和手动选择 Chromium 浏览器。
- 新增登录状态自动读取、失效提示和重新登录流程，保留手工 Cookie 备用入口。

### 调整

- 列表检索组合改为固定 3 路并发，每个组合内部仍按页顺序完整抓取。
- 请求层改用线程本地 Session 复用连接，保留原有三次请求重试。
- 并发失败的列表组合自动串行整组补抓，补抓仍失败则终止任务。
- 详情五路并发增加串行补抓，最终失败时保留列表数据并标记人工核对。

- Excel 写入前按日期升序排列，写入后对主表全局升序并重新编号；筛选后表保持相同顺序。

- 自动选择浏览器的优先级调整为 Chrome、Brave、Edge，以提高自动登录稳定性。
- 打包时显式排除未使用的 NumPy/MKL 可选依赖，避免发布包异常膨胀。
- 使用干净依赖环境重新构建并验证启动，v1.1.0 Windows ZIP 约 15.9 MB，且不含登录状态、业务结果和备份文件。

- 将列表和详情阶段调整为宽口径写入：不再因标题、摘要或详情正文初步判断不相关而提前丢弃候选项目。
- 原始表负责尽量保留候选，筛选后表负责严格业务终筛，降低初筛误删风险。
- 筛选前会去除标题和正文中的空白字符，避免“结构监测 服务”等写法漏匹配。
- 公告详情页改为固定 5 路并发获取，完成后按原列表顺序解析和写入。

### 修复

- 修复新版乙方宝登录后仅写入 `Admin-Token`、未写入 `yfbSite.session.id` 时被误判为未登录的问题。
- 修复专用浏览器启动异常时可能残留后台进程的问题。
- 修复生成筛选后 Excel 时，删除穿过日期合并区域的行会触发 `KeyError: (行号, 2)` 并中断的问题。
- 修复“绿建验收”简称未被保留的问题。
- 修复“结构监测 服务”因空格和词表缺失被误删的问题。
- 明确记录“物探”类项目目前尚未纳入正式保留规则，需按业务边界确认。

### 计划
- 根据实际使用结果继续补充公告格式和建设单位识别规则。
- 完善异常日志、可配置关键词和地区设置。
- 增加发布前的自动化检查。

## [1.0.0] - 2026-06-27

### 新增

- Windows Tkinter 桌面操作界面。
- 乙方宝 Cookie 输入、Excel 选择、时间范围选择和演练模式。
- 山东济南、莱芜地区限定。
- 监测、水土保持、测绘、测量四组关键词检索。
- 全文/标题与智能/精准四种检索组合。
- 公告列表、详情正文及部分国铁采购公告的补充获取。
- 项目标题标准化去重和 Excel 已有项目去重。
- 建设单位、资格要求、报名时间、投标截止时间提取。
- “明确相关”和“疑似相关”分类。
- 常见无关项目排除规则和重点业务词保护规则。
- Excel 主表与“公告详情”工作表双向超链接。
- 写入前自动备份、列宽和行高自动调整。
- PyInstaller Windows 打包脚本及中文使用说明。
- GitHub README、项目日志和版本库排除规则。
- 不含真实业务数据的公开发布模板。

### 调整




- 源码运行和 Windows 打包统一使用根目录 `.venv`。
- 固定 openpyxl、PyInstaller 和构建钩子版本，提高构建可复现性。
- 打包脚本会自动创建隔离环境，不再依赖全局 Conda 软件包。
- Windows 发布 ZIP 从约 247MB 缩减到约 13MB。
- README 与 MD/TXT 说明书明确区分 EXE 用户和源码开发环境。

### 修复

- 补充从公告正文和项目标题识别建设单位的规则。
- 修复长文本直接堆积在主表中导致难以浏览的问题。
- 修复 Markdown 说明书在部分 Windows 环境下无法直接打开的问题，增加 TXT 回退。
- 修复打包脚本未复制 TXT 说明书的问题。
- 修复空白模板首次写入时数据行样式来源不正确的问题。
- 改进变更公告、延期公告和重复标题的合并逻辑。

### 已知问题

- Cookie 过期后必须重新登录获取。
- 网站接口变化可能导致抓取失败。
- 少数公告无法完整提取资质、单位或时间，需要人工核对。
- 高召回率检索会保留部分疑似结果。
