# 乙方宝招标信息爬取与整理说明书

## 适用场景

本工具用于从乙方宝/千里马企业站搜索招标信息，并把结果追加到 Excel 表格中。当前配置面向：

- 地区：山东济南、莱芜
- 关键词：监测、水土保持、测绘、测量
- 输出文件：`2026-乙方宝招标信息统计.xlsx`
- 输出结构：主表放摘要，`公告详情` sheet 放资质和公告长文本

## 文件组成

建议打包时至少包含这些文件：

- `yfb_bid_spider.py`：主爬虫脚本
- `2026-乙方宝招标信息统计.xlsx`：Excel 模板/数据表
- `requirements.txt`：固定版本的运行依赖
- `requirements-dev.txt`：固定版本的打包依赖
- `乙方宝爬虫使用说明.md`：本说明书

可选文件：

- `2026-乙方宝招标信息统计-审核清理版.xlsx`：人工审核后的清理版结果
- `审核清理删除项目清单.txt`：清理版删除记录

## 环境要求

### 直接使用 EXE

- Windows 10/11
- 能访问乙方宝网页
- 已登录乙方宝账号
- 不需要安装 Python
- 不需要创建或激活 `.venv`

### 从源码运行或重新打包

- Python 3.10 或更高版本，推荐 Python 3.12
- 当前发布构建验证环境：Python 3.12.7
- 运行依赖：openpyxl 3.1.5
- 打包依赖：PyInstaller 6.21.0

项目使用上一级目录中的 `.venv` 隔离依赖。首次配置时，在项目根目录运行：

```powershell
python -m venv .venv
.\.venv\Scripts\python.exe -m pip install -r ".\02_源码和说明\requirements.txt"
```

无需激活虚拟环境，后续命令直接调用 `.venv\Scripts\python.exe` 即可。

## Cookie 获取方法

1. 打开乙方宝页面并登录。
2. 进入搜索页：`https://qiye.qianlima.com/new_qd_yfbsite/#/infoCenter/search`
3. 按 `F12` 打开开发者工具。
4. 点击上方 `网络`。
5. 在页面里随便搜索一次。
6. 在网络请求里找到类似 `queryZBInfo` 的请求。
7. 点开该请求，找到 `请求标头`。
8. 复制里面的 `Cookie` 整行内容。

注意：Cookie 是登录凭证，不要发给无关人员。Cookie 过期后需要重新复制。

## 打包成 EXE

在 Windows 电脑上双击：

```text
打包成EXE.bat
```

脚本会自动使用项目根目录的 `.venv`，安装 `requirements-dev.txt` 中固定版本的依赖，然后运行 PyInstaller。手动打包：

```powershell
cd "项目根目录"
python -m venv .venv
.\.venv\Scripts\python.exe -m pip install -r ".\02_源码和说明\requirements-dev.txt"
cd ".\02_源码和说明"
..\.venv\Scripts\python.exe -m PyInstaller --noconfirm --clean --windowed --name "乙方宝招标信息爬取工具" --hidden-import yfb_bid_spider yfb_spider_app.py
```

打包完成后，程序在：

```text
dist\乙方宝招标信息爬取工具
```

把整个文件夹压缩为 ZIP 后发布。使用者解压并运行 EXE 即可，不需要 Python 或 `.venv`。

## 当前搜索逻辑

关键词依次搜索：

- 监测
- 水土保持
- 测绘
- 测量

地区限制：

- 济南
- 莱芜

检索组合共 4 种：

- 全文检索 + 智能检索
- 全文检索 + 精准检索
- 标题检索 + 智能检索
- 标题检索 + 精准检索

对应脚本参数：

- `filterCondition=1`：全文检索
- `filterCondition=2`：标题检索
- `searchType=1`：智能检索
- `searchType=2`：精准检索

## 结果判断逻辑

脚本会把结果分成：

- `明确相关`：标题直接命中监测、水土保持、测绘、测量、地形图、规划核实、房产实测、国土变更调查等业务词。
- `疑似相关`：标题不明显，但正文或摘要里出现相关业务词。

为了减少误报，脚本已经内置明显无关排除词，例如：

- 车位使用权
- 房间招标
- 招租
- 废旧资产
- 机械密封
- 加工件
- UPS 系统采购
- 测绘仪器无人机采购
- 设备与材料采购
- 住宅、储藏室转让
- 华电莱城发电厂加工件类公告

同时保留一些容易误删但可能相关的业务词，例如：

- 水土保持
- 测绘服务
- 国土测绘
- 地形图
- 规划核实
- 房产实测
- 国土变更调查
- 用地预审
- 确权登记
- 土地复垦
- 竣工测量
- 变形监测
- 基坑监测

## 推荐使用流程

1. 先运行爬虫：`.\.venv\Scripts\python.exe ".\02_源码和说明\yfb_bid_spider.py" --days 31`
2. 打开 Excel，先筛选 `相关性=明确相关`。
3. 再筛选 `相关性=疑似相关` 做人工复核。
4. 对大范围爬取结果，建议再做一次审核清理。
5. 保留脚本自动生成的备份文件，避免误删后无法恢复。

## 常见问题

### 认证失败

现象：提示 `认证失败，无法访问系统资源`。

处理：重新登录乙方宝，复制新的 Cookie。

### Excel 保存失败

现象：`PermissionError`。

处理：关闭正在打开的 Excel 文件后重新运行。

### 结果里仍然有无关项目

原因：为了尽可能全面，全文检索和疑似相关会保留一些边界项目。

处理：筛选 `相关性=疑似相关`，人工检查；必要时把新发现的噪声标题词加入脚本的 `NOISE_TITLE_TERMS`。

### 漏掉项目

可能原因：乙方宝接口没有返回、Cookie 权限变化、关键词没有覆盖、地区筛选不一致。

处理：先在网页手动搜索确认，再把标题或关键词补充到脚本逻辑。

## 维护位置

主要改这些位置：

- `SEARCH_KEYWORDS`：关键词
- `AREA_IDS`：地区
- `PROTECTED_TITLE_TERMS`：需要优先保留的业务词
- `NOISE_TITLE_TERMS`：明显无关排除词
- `STRICT_NOISE_TITLE_TERMS`：即使包含业务词也要排除的强噪声
